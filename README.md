# cookwithlove

**CookWithLove** is a web-based platform designed to manage and display cooking courses. This project features a full-stack architecture using PHP for server-side logic and MySQL for data management.

## 🚀 Features
* **User Authentication**: Secure Registration and Login system using password hashing.
* **Dynamic Content**: Fetches and displays available cooking courses directly from a database.
* **Database Integration**: Centralized database connection for easy maintenance.
* **Clean UI**: Simple and intuitive interface for browsing courses.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3
* **Backend**: PHP 
* **Database**: MySQL

## 📂 Project Structure
* `db.php` - Database connection configuration.
* `register.php` - User registration logic and form.
* `login.php` - User login logic and session management.
* `fetch_courses.php` - Logic to retrieve course data from the database.
* `index.html` - Main landing page of the application.
